# State Transition

Объект Booking может находиться в следующих состояниях:

pending
confirmed
cancelled

Переходы, которые необходимо покрыть тестами:

1. pending → confirmed
   Проверяет успешное создание и подтверждение бронирования.

2. pending → cancelled
   Проверяет отмену бронирования до подтверждения.

3. confirmed → cancelled
   Проверяет отмену уже подтверждённого бронирования.

4. confirmed → confirmed
   Проверяет защиту от повторного подтверждения.

5. cancelled → confirmed
   Недопустимый переход. Система не должна позволять восстановить отменённое бронирование без создания нового.

6. cancelled → cancelled
   Проверяет защиту от повторной отмены.
