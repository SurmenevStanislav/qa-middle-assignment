# Найденные дефекты

## BUG-001. Отсутствие валидации при попытке загрузки файла без выбора файла

Severity: Medium

Страница: File Upload (/upload)

### Шаги воспроизведения

1. Открыть страницу File Upload.
2. Не выбирать файл.
3. Нажать Upload.

### Ожидаемый результат

Система должна отобразить сообщение:

- «Файл не выбран»
  или
- «Необходимо выбрать файл для загрузки».

### Фактический результат

Отображается страница Internal Server Error.

Во вкладке Network запрос POST /upload возвращает статус 500 Internal Server Error.
